# SmartSeat 🚗👶
**An IoT-Based Safety System for Preventing Child Abandonment in Vehicles**

## 📌 Project Overview
SmartSeat is an innovative IoT safety solution designed to tackle "Forgotten Baby Syndrome" and prevent vehicular heatstroke. By integrating hardware sensors with real-time cloud logic and cellular/mobile alerts, the system serves as an automated, multi-tiered fail-safe for parents and caregivers.

### Key Features:
* **Real-Time Occupancy Sensing:** Uses Force-Sensitive Resistors (FSR) to detect child presence.
* **Environmental Monitoring:** Continuously tracks vehicle interior temperature and humidity.
* **Intelligent Distance Alerts:** Triggers a progressive alarm sequence if the driver moves away from the vehicle while the seat is occupied.
* **Multi-Tiered Notification System:** Local buzzer/LED alerts escalating to smartphone notifications and automated emergency SMS.

---

## 🛠️ System Architecture & Tech Stack
* **Microcontroller:** ESP32 (Dual-core processing, low-power modes)
* **Languages:** C / C++ (Firmware architecture)
* **IoT Protocols & Cloud:** WiFi, Bluetooth (BLE), Blynk IoT Platform
* **Hardware Components:** FSR Pressure Sensors, DHT11 Temperature & Humidity Sensor, Active Buzzers, Logic Gates
